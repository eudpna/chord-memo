# Chord Memo

[chordshortcut.nyaw.net](https://chordshortcut.nyaw.net/)

ギターおよびウクレレコード譜の作成ツールです。

[@tombatossals/chords-db](https://github.com/tombatossals/chords-db) を使用しています。

オンコード(スラッシュコード)は部分的にしか対応できていません。オンコード以外は大体カバーしてると思います。


### How to run

```sh
yarn install
yarn build
yarn start
```

built on Next.js
