# Chord Memo

[chordmemo.nyaw.net](https://chordmemo.nyaw.net/)

ギターおよびウクレレコード譜の作成ツールです。

[@tombatossals/chords-db](https://github.com/tombatossals/chords-db) を使用しています。

オンコード(分数コード)は僅かしか対応できていません。

ソースコードが未整理で汚いです。ごめんなさい。


### How to run

```sh
yarn install
yarn build
yarn start
```

built on Next.js
